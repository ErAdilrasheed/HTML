Tables- The table tag is used to define a table in html, it is used to format and display tabular data. Some tags that are used in table.
tr tag - used to display table row.
td tag - used to display table data.
th tag - used in place of table data for displaying table headers.
We can define as many rows as we want. to add a caption in a table we use caption tag inside table.
thead tag - used to wrap table head (caption & tr with th).
tbody tag - used to wrap the table body.

HTML Forms

An html form is used to collect input from user form tag is used for the same.

There are different form element for different kind of user inputs.
Input Elements- Can be of type text, checkbox, radio, button and Submit We also have a file type.
Textarea Elements - Defines a multi line text input, Cols and rows attributes can be used to size the textarea.
Select Elements - Defines a drop down list.
Note - you don not need to remember all kind of tags , you will automatically memorize them with practice.
