HTML SEO

HTML developers can implement SEO using the following techniques:
  1. Set the title very nice & to the point
  2. set the meta description.
  3. meta name="description" content=" ...">
  4. set a nice URL Slug
  5. Set the meta keywords tag.
  6. set the meta author tag.
  7. meta name="author" content = "Adil">
  8. Set a favicon
  9. Compress images & other resources
  10. Remove unused HTML/CSS & Js files + Compress them
  11. Add alt test to images
